# Attributions

OPPi is built on and inspired by open-source agent projects and modern coding-agent workflows.

## Core upstreams and references

- [Pi / pi-coding-agent](https://github.com/badlogic/pi-mono) — upstream runtime, SDK, extension model, TUI primitives. License: MIT.
- [Oh My Pi](https://github.com/can1357/oh-my-pi) — native tooling architecture, plugin ideas, marketplace patterns, memory and workflow experiments. License: MIT.
- [OpenAI Codex CLI](https://github.com/openai/codex) — review flow, app-server concepts, permission UX inspiration, image-generation behavior, Rust architecture references. License: Apache-2.0.
- [Caveman](https://github.com/juliusbrussee/caveman) by Julius Brussee — prompt-strategy reference for OPPi system prompt experiments and possible future prompt architecture. License: MIT.
- [Meridian](https://github.com/rynfar/meridian) — optional local Anthropic-compatible bridge backed by the official Claude Code SDK for Claude subscription access. License: MIT.
- [beautiful-mermaid](https://www.npmjs.com/package/beautiful-mermaid) — Mermaid-to-ASCII rendering used by OPPi's `render_mermaid` tool. License: MIT.

## Behavioral inspiration

OPPi is also informed by user experience patterns from modern AI coding CLIs, including our favorite orange CLI tool. No source from non-open/proprietary leaks should be copied or inspected for implementation.
