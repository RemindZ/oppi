# @oppiai/tui

Future custom terminal harness.

Initial OPPi should run as a Pi package inside Pi. Build this only after the OPPi Pi package is useful and stable.

Planned responsibilities:

- OPPi-owned layout
- low-flicker streaming renderer
- Codex-style spinner/catch-up streaming behavior
- multi-agent tabs
- better status surfaces
- eventual app-server client mode
